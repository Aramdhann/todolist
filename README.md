# Todolist Web

### About:

This is a To Do List web using React Js with Vite. User can add the list what they want to do and also the description for detail. User also can edit and delete to do list

### Tech Stack:

-   React JS
-   TailwindCSS
-   NodeJs
-   React Router Dom
-   React Redux
